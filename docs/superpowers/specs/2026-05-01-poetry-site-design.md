# 诗词天地 — 网站设计规范

**日期：** 2026-05-01  
**状态：** 已确认

---

## 概述

一个展示中国古典与现代诗词的静态网站，面向普通大众，提供沉浸式阅读体验。核心亮点是根据诗词主题自动生成的意境场景动画，以及全屏沉浸式的诗词详情页。

---

## 目标与约束

- **目标用户：** 普通大众（休闲浏览，轻松欣赏）
- **诗词类型：** 古典诗词（唐诗宋词元曲）+ 现代诗歌
- **视觉风格：** 双主题——首页和诗人页采用现代简约浅色风格（白底、大留白、黑白灰调）；诗词详情页切换为深色沉浸主题（深色背景 + 意境场景动画），形成"浏览"与"沉浸"两种体验的自然对比
- **核心体验：** 全屏意境动画 + 诗词沉浸式阅读
- **部署方式：** 纯静态，无需服务器

---

## 技术架构

```
前端框架：Vue 3 + Vite
动画：CSS Transitions + Canvas API
数据：本地 JSON 文件
部署：Vercel 或 GitHub Pages（免费）
```

无后端、无数据库。所有诗词内容存于本地 JSON 文件，新增诗词只需编辑 JSON。

## 页面结构

| 路由 | 页面 | 描述 |
|------|------|------|
| `/` | 首页 | 搜索栏 + 诗词卡片网格 |
| `/poem/:id` | 诗词详情页 | 全屏沉浸式，意境动画背景 |
| `/poet/:id` | 诗人页 | 头像 + 生平 + 作品列表 |

**导航栏**（固定顶部）：网站名 · 探索 · 诗人 · 搜索图标

### 首页

1. 顶部搜索栏（支持诗名、诗人、关键词搜索）
2. 分类标签（全部 / 唐诗 / 宋词 / 现代诗）
3. 诗词卡片网格（卡片内容：诗名、诗人、朝代、首句预览）

### 诗词详情页

1. 全屏 Canvas 意境动画背景（根据 `scene` 字段切换）
2. 诗句叠加动画背景上，逐句淡入显示
3. 向下滚动进入释义区：
   - 白话翻译
   - 逐句解析
4. 底部诗人简介卡片（可点击跳转诗人页）

### 诗人页

1. 诗人头像（插画风）+ 朝代、流派标签
2. 生平简介文字
3. 该诗人全部诗词卡片列表

## 意境场景系统

`scene` 字段将诗词与场景模块绑定。`SceneCanvas` 组件接收 `scene` 参数，动态加载对应场景模块并在 Canvas 上运行动画。

| scene 值 | 场景名称 | 动画效果 | 适用主题 |
|----------|----------|----------|----------|
| `moonlit-night` | 月夜 | 星星闪烁、月亮光晕、人物剪影 | 思乡、月亮、夜晚 |
| `spring-morning` | 春晨 | 花瓣飘落、鸟鸣、晨光粒子 | 春天、自然、生机 |
| `river-flow` | 江河 | 水波纹动态、远山飘雾 | 送别、旅途、江河 |
| `rain` | 烟雨 | 雨丝粒子下落、水面涟漪 | 清明、惆怅、离愁 |
| `autumn` | 秋意 | 落叶飘落粒子、暖橙色调 | 秋天、感怀、岁月 |

新增场景只需在 `src/scenes/` 目录下添加一个导出 `init(canvas)` 和 `destroy()` 的 JS 模块。

```
src/
├── components/
│   ├── NavBar.vue          # 顶部导航栏
│   ├── SearchBar.vue       # 搜索框 + 分类标签过滤

├── views/
│   ├── HomeView.vue        # 首页
│   ├── PoemView.vue        # 诗词详情页

├── scenes/
│   ├── moonlit-night.js    # 月夜动画模块
│   ├── spring-morning.js   # 春晨动画模块
│   ├── river-flow.js       # 江河动画模块
│   ├── rain.js             # 烟雨动画模块
│   └── autumn.js           # 秋意动画模块
└── data/
    ├── poems.json          # 诗词数据
    └── poets.json          # 诗人数据
```

### 关键组件接口

**SceneCanvas.vue**
```
Props: scene: string
事件：无
职责：根据 scene 值动态 import 对应场景模块，在 Canvas 上运行动画，组件卸载时调用 destroy() 清理
```

**PoemReader.vue**
```
Props: lines: string[], delay: number（每句间隔毫秒）
职责：按 delay 间隔依次将每句诗的 opacity 从 0 过渡到 1
```

## 错误处理

- 搜索无结果：显示"暂无结果"提示，建议用户换一个关键词
- 诗词 ID 不存在（`/poem/invalid-id`）：重定向到首页
- 场景模块加载失败：降级为纯色渐变背景，不影响诗词阅读
- 诗人头像图片加载失败：显示文字头像（取名字第一个字）
